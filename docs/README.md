# GIS Long Bình — Docs

**Bắt đầu tại:** [PROJECT.md](./PROJECT.md) — tổng quan dự án, khái niệm, luồng frontend, trạng thái triển khai.

**API từng module:** [modules/](./modules/)

---

Cấu trúc:

```
docs/
├── PROJECT.md          ← tài liệu tổng (đọc file này trước)
├── modules/            ← một file / module (API, debug, frontend)
├── phases/             ← lộ trình triển khai
├── appendix/           ← tham chiếu sâu (field types, import, API conventions)
├── architecture-v3.1.md
└── data-model.md
```

Khi thêm/sửa chức năng → cập nhật `docs/modules/<module>.md`.
